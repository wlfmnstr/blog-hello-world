# System Features Overview

Our knowledge base comes with a rich set of features designed to enhance your documentation experience.

## Content Management

### Visual Editor

The Tina CMS integration provides a powerful visual editor that makes content creation accessible to everyone:

- Rich text formatting
- Image uploads and management
- Real-time preview
- Intuitive interface

### Direct File Editing

Technical users can work with the raw MDX files:

```bash
# Clone the repo
git clone https://github.com/your-org/knowledge-base.git

# Make your changes
vim src/content/docs/new-guide.mdx

# Commit and push
git add .
git commit -m "Add new guide"
git push
```

## Search Capabilities

Press **Cmd/Ctrl+K** to open the search dialog. Our Pagefind integration provides:

- Instant results as you type
- Full-text search across all content
- Keyboard navigation
- Highlighting of matched terms

## Theme Support

Switch between light and dark modes with the theme toggle. Your preference is saved automatically and persists across sessions.

## MDX Components

Use custom components in your content:

- Callouts for important information
- Code blocks with syntax highlighting
- Tabs for organizing content
- And more!

## Automatic Deployment

Every commit to the main branch triggers an automatic deployment via GitHub Actions. Changes go live in minutes.
